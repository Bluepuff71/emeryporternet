Tamagotchi

Complete

A virtual pet game built in Javatech, a custom Java game engine.

Latest Release

Origin

Tamagotchi was my submission for the final project of CS4365: Software Engineering II.

Overview

Tamagotchi allows you to take care of and communicate with a virtual pet. Your pet must eat, drink, be cleaned, and be played with in order to stay happy and alive.

Systems

The game was built upon JavaTech, a rudimentary custom game engine written by myself.
