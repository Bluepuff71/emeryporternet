Superfist

A silly 3D fighting game built in the Unity Engine

Latest Release

Origin

Superfist is a game I made during HackwestTX 2020.

Overview

Superfist is a 3D fighting game built in the Unity Engine about ordinary people with extraordinary punching ability.

Systems

The fighting system was bought from the unity asset store and heavily modified by me to support Unity's new input system, block sliding, and ragdolling.
